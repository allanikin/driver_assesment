SRBC Waste Services Driver Competency Assessment PWA — Version 4

This version replaces the generic vehicle-condition checklist with an assessor-focused
observed walkaround assessment.

The assessor follows the driver around an interactive refuse collection vehicle and records
whether the driver competently performed each DVSA HGV walkaround item.

Included:
• all 27 items from the DVSA HGV walkaround check diagram
• five additional Waste Services checks
• interactive cab, front, nearside, rear, offside and body zones
• Competent, Coaching, Item omitted, Unsatisfactory and Not applicable outcomes
• predefined assessor observations and optional narrative
• separate recording when the driver identifies a genuine vehicle defect
• automatic pre-drive performance report and PDF table
• existing driving assessment, GPS, report editing and digital signatures

Deployment:
Upload all files to the GitHub Pages publishing folder. This build uses cache version v4.
After deployment, open the site using ?v=4 once to bypass an older Safari cache.
